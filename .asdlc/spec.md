# Overview

A todo app where people add tasks, mark them done, and delete them. The interface is a web page that stores tasks in a database.

# Personas

- User — adds tasks, checks them off, and removes them from the list.

# Features

- A user types a task description and adds it to their list.
- The list shows all tasks, including which ones are done.
- Users mark a task as complete by clicking it.
- Users delete tasks they no longer need.
- Tasks persist so they're still there when the user comes back later.
